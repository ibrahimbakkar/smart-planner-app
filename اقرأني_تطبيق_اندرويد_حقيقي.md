# تطبيق أندرويد حقيقي 100% (مش مجرد PWA) — مجاني بالكامل

الملف ده فيه مشروع تطبيق أندرويد حقيقي (اسمه تقنيًا Capacitor project). الفرق عن الطريقة اللي عملناها قبل كده (PWABuilder):

- ✅ منبهات المهام هتشتغل حتى لو التطبيق **مقفول تمامًا** (مش بس مصغّر) — لأنها بقت مجدولة جوه نظام أندرويد نفسه (AlarmManager)، مش جوه المتصفح.
- ✅ المنبهات بتفضل شغالة حتى بعد ما تعيد تشغيل الموبايل.
- ✅ تطبيق حقيقي، بيتبني وبيتحول لملف APK عادي عن طريق GitHub (مجاني 100%، مفيش أي اشتراك).

## الخطوات

### 1. اعمل Repository جديد على GitHub (منفصل عن اللي فات)
1. روح github.com → New repository
2. اسم زي: `smart-planner-app`
3. Public → Create

### 2. ارفع كل الملفات اللي في الملف المضغوط ده
1. فك ضغط الملف المضغوط ده على جهازك
2. جوه الـ repository الجديد: Add file → Upload files
3. اسحب **كل المجلدات والملفات** اللي جوه الملف المضغوط (بما فيها مجلد `.github` و `android` و `www` — لو المتصفح مسمحش برفع مجلد `.github` مباشرة لإنه مخفي، استخدم أداة زي GitHub Desktop، أو ارفعها بطريقة "git" العادية لو عندك خبرة، أو قولي وهساعدك بطريقة تانية)
4. Commit changes

### 3. شغّل عملية البناء (تلقائية)
بمجرد ما ترفع الملفات، GitHub هيشغّل "Actions" تلقائيًا (فيه ملف جاهز بيعمل كده: `.github/workflows/build-apk.yml`).

1. روح لتبويب **"Actions"** فوق في صفحة الـ repository
2. هتلاقي عملية شغالة اسمها "Build Android APK" (دايرة صفرا بتلف = لسه شغالة، تستغرق حوالي 3-6 دقايق)
3. استنى لحد ما تخضر ✅

### 4. حمّل ملف الـ APK
1. دوس على العملية اللي خلصت (اللي عليها ✅)
2. انزل تحت لقسم **"Artifacts"**
3. هتلاقي ملف اسمه `smart-planner-apk` — دوس عليه يحمّل ملف مضغوط صغير
4. فك ضغطه، هتلاقي جواه `app-debug.apk` — ده تطبيقك الحقيقي!
5. انقله للموبايل ونصبه (هيطلب "السماح من مصادر غير معروفة" أول مرة، عادي جدًا)

### 5. لما تعمل أي تعديل مستقبلي
كرر خطوة "Upload files" برفع الملفات المحدّثة بس (زي `www/index.html` الجديد)، والعملية هتشتغل تلقائيًا تاني وتطلعلك APK محدّث في نفس المكان.

## ملاحظات مهمة وصريحة

- ده تطبيق **Debug** (تجريبي)، يعني هيشتغل بنسبة 100% وتقدر تستخدمه عادي، بس لو حبيت تنشره على Google Play لاحقًا محتاج خطوة إضافية اسمها "Signing" — قولي لو وصلت للمرحلة دي وهساعدك.
- أول مرة تفتح التطبيق، هيطلب إذن الإشعارات (Notifications) — لازم توافق عشان المنبهات تشتغل.
- بعض أجهزة أندرويد (خصوصًا Xiaomi وHuawei وOppo) بتوقف التطبيقات في الخلفية بشكل عدواني لتوفير البطارية. لو المنبهات مبقتش ترن بعد فترة، روح لإعدادات الموبايل → التطبيقات → المنظم الزمني الذكي → البطارية → اختار "بدون قيود" (Unrestricted) عشان النظام مايوقفش التطبيق.
- المزامنة (☁️) والذكاء الاصطناعي (🔑) هيفضلوا شغالين بالظبط زي ما كانوا، معملناش فيهم أي تغيير.
